0. 프로젝트 개요
이 문서는 해외영업 직무 포트폴리오용 글로벌 바이어 CRM 대시보드의 전체 기능 스펙을 정의합니다. 제조/무역 B2B 및 IT/SaaS 분야의 해외영업 실무를 반영하여 설계되었습니다.
📌 프로젝트 핵심 가치
① 해외영업 실무 프로세스를 직접 설계·구현한 역량 증명
② 데이터 기반 의사결정 능력 (수출 통계, Deal 파이프라인 시각화)
③ AI 활용 업무 자동화 감각 (영문 이메일 자동 생성)
④ 풀스택 개발 능력을 비개발 직무에 융합한 차별화 포인트

3. 화면 구성 & 네비게이션
총 5개 메인 화면으로 구성됩니다. 좌측 사이드바 + 우측 콘텐츠 레이아웃을 기본으로 합니다.
3-1. 대시보드 홈 (/)
상단 4개 KPI 카드: 총 바이어, 파이프라인 총액, 이번 달 미팅, Win Rate
좌: 국가별 바이어 도넛 차트 / 우: 최근 Deal 현황 리스트
하단: 이번 주 예정 미팅 타임라인
3-2. 바이어 관리 (/buyers)
상단 검색바 + 국가/산업/등급 필터 태그
카드형 또는 테이블형 뷰 전환 버튼
바이어 카드: 국기 이모지, 회사명, 담당자, 최근 미팅일, 등급 뱃지
클릭 시 상세 슬라이드 패널 (오른쪽에서 슬라이딩)
3-3. Deal 파이프라인 (/pipeline)
5개 컬럼 Kanban 보드: Prospecting / Contacted / Negotiating / Contracting / Closed
각 컬럼 상단에 Deal 수 + 금액 합산 표시
Deal 카드: 바이어명, 품목, 금액, D-day, 담당자 아바타
드래그앤드롭으로 스테이지 이동, 이동 시 자동 날짜 기록
3-4. 미팅 관리 (/meetings)
월 캘린더 뷰 + 리스트 뷰 전환
미팅 클릭 시 상세 모달: 안건, 참석자, 메모, Action Item
'미팅 요약 AI 생성' 버튼 → Claude가 요약 + Follow-up 이메일 초안 생성
3-5. AI 이메일 생성기 (/email)
좌: 입력 패널 (바이어 선택, 이메일 목적, 톤, 추가 메모)
우: 실시간 AI 생성 결과 + 복사 버튼
하단: 저장된 템플릿 목록

5. 바이브코딩 프롬프트 전략
Claude / Cursor / v0 등을 활용한 효율적인 바이브코딩 순서와 핵심 프롬프트 패턴입니다.
5-1. 추천 개발 순서
프로젝트 세팅: React + Vite + Tailwind + shadcn/ui 설치
레이아웃 & 라우팅: 사이드바 + 5개 페이지 라우터 구성
바이어 관리 모듈: 등록 폼 → 목록 → 필터 → 상세 페이지 순서
Kanban 파이프라인: react-beautiful-dnd 기반 드래그앤드롭
대시보드 차트: KPI 카드 → 도넛 차트 → 라인 차트
AI 이메일 생성기: Claude API 연동 + 스트리밍 응답 처리
미팅 관리: 캘린더 뷰 + 타임라인 + Action Item
샘플 데이터 투입 + Vercel 배포 + 포트폴리오 문서화
5-2. 핵심 프롬프트 템플릿
📝 초기 세팅 프롬프트
복사해서 Claude/Cursor에 붙여넣기
나는 해외영업 포트폴리오용 글로벌 바이어 CRM 웹앱을 만들고 있어.
React 18 + Vite + Tailwind CSS + shadcn/ui 기반으로 프로젝트를 세팅해줘.
폴더 구조는 /components, /pages, /store, /hooks, /utils 로 잡아줘.
사이드바는 대시보드, 바이어 관리, Deal 파이프라인, 미팅 관리, 이메일 생성기 5개 메뉴로 구성해줘.
전체 톤은 B2B SaaS 스타일의 다크 사이드바 + 화이트 콘텐츠 레이아웃으로 해줘.
🤖 AI 이메일 생성기 프롬프트
Claude API 연동 컴포넌트 생성
Claude API를 사용해서 영문 비즈니스 이메일을 생성하는 컴포넌트를 만들어줘.
입력값: 바이어 회사명, 담당자 이름, 이메일 목적(첫 접촉/견적 요청/팔로업/감사), 추가 메모
톤 선택: Formal / Friendly / Urgent 3가지
응답은 스트리밍으로 받아서 실시간으로 텍스트가 나타나는 효과를 주고,
생성 완료 후 '복사' 버튼과 '템플릿으로 저장' 버튼을 보여줘.
